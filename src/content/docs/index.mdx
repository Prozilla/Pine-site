<About title="About Pine">
	Pine is an abbreviation for **P**rozilla's game eng**ine**.
	Pine implements the [LWJGL](https://www.lwjgl.org/) library to enable cross-platform access to native API's like OpenGL, Vulkan, OpenAL, OpenCL and more.
	It provides a high-level and developer-friendly framework on top of the native libraries that LWJGL provides.
